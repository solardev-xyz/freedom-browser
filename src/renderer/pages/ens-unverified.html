Resolution not cross-checked

Only one RPC server answered this ENS resolution, so the browser couldn't cross-check the answer against independent sources. The destination may be correct, but a single malicious server could have chosen it — we can't tell.

You can disable this warning globally in Settings (not recommended on public Wi-Fi). Running your own Ethereum node as a custom RPC is the strongest option — your own node becomes the single trusted source and this warning stops appearing.
